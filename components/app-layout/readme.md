### Layout 通用父布局

### 使用方式

在 ``script`` 中引用组件 

```javascript
import { AppLayout } from '@/components';
export default {
    components: { AppLayout }
}
```

在 ``template`` 中使用组件

```html
<app-layout></app-layout>

```
